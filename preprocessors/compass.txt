Compass Sprites: 
Use @import "your-folder-name/*.png"; to import all .png files in your sprite folder and generate a sprite sheet
Use @include all-<your-sprite-map>-sprites; to create CSS rules for all images inside your sprite folder
To display hover states, add the new images to your sprites folder ЦЦ with the same file names, and make sure the file names end in _hover.png
$<your-sprite-map>-sprite-dimensions: true; Ц Outputs the width and height values for each sprite in the CSS

Production Stylesheets:
